隐私说明

看见你的电脑，不带走你的数据。

CoreRobin 是本机桌面应用。监控、扫描、历史与偏好用于这台电脑上的界面和诊断；重要权限与永久删除始终需要你确认。

最后核对：CoreRobin 0.0.4 · 2026-07-17在 GitHub 阅读

不会做什么

  • CoreRobin 0.0.4 不包含产品遥测、崩溃上报 SDK 或账户同步服务。
  • 监控、文件扫描、进程、连接、历史和偏好不会被 CoreRobin 上传到我们的服务器。
  • 扫描不读取文件内容，也不会在扫描期间移动、修改或上传文件。
  • CoreRobin 不会自动结束进程、卸载应用或删除文件。

会在本机保存什么

实时状态

CPU、内存、磁盘、网络、应用、进程、启动项和连接的本机快照，用于当前界面和诊断。

历史与操作记录

历史可保留 1、7 或 30 天，也可关闭或随时清空。应用名称只有在你允许后才会跨重启保存。

空间扫描结果

只读取名称、大小、时间、位置和文件类型等元数据。结果最多保留 7 天，超过 24 小时会标记为较旧。

偏好

语言、动态效果、通知、历史保留、后台启动和 Robin 伙伴偏好用于恢复你的选择。

完全磁盘访问权限

macOS 会保护邮件、信息和其他应用数据等位置。开启权限后，CoreRobin 可以更完整地读取系统磁盘的元数据，让空间地图少遗漏受保护的位置。

  • 权限由 macOS 管理，可随时在系统设置中关闭。
  • 不授权时仍可扫描可访问区域，受保护位置会被明确标记。
  • 它不会让扫描读取文件内容，也不会让应用在未确认时删除文件。

清除数据、卸载与反馈

可以在“设置 → 关于与支持”清除全部本机数据，也可以只在“记录”页清空历史、在清理页清除保存的扫描结果。完整卸载前，请先在设置中关闭登录后启动，从状态栏选择“退出应用”，并在系统工具中确认没有 CoreRobin 进程。macOS 用户还应在“系统设置 → 隐私与安全性 → 完全磁盘访问权限”中关闭 CoreRobin。

macOS

~/Library/Application Support/com.corerobin.monitor/
WebView 或偏好残留可能还包括 Library/WebKit、Library/Caches 和 Library/Preferences 中使用相同标识的项目；登录启动项为 Library/LaunchAgents/core-robin.plist。

Windows

%APPDATA%\com.corerobin.monitor\
%LOCALAPPDATA%\com.corerobin.monitor\
登录启动项是当前用户 Run 与 StartupApproved 中名为 core-robin 的值。

Linux

${XDG_DATA_HOME:-~/.local/share}/com.corerobin.monitor/
${XDG_CONFIG_HOME:-~/.config}/autostart/core-robin.desktop

不同系统或 WebView 版本可能只创建其中一部分。公开 Issue 对任何人可见；请从“关于与支持”复制版本和诊断摘要，先预览并隐藏用户名、路径、IP 地址、访问令牌和截图中的个人信息。安全漏洞请使用私密漏洞报告。
