macOS 首次打开

从官方 DMG 安装，然后正常启动。

macOS DMG 已使用 Developer ID 签名、经过 Apple 公证、装订票据并通过 Gatekeeper 校验。Apple Silicon 已完成实机验证；Intel 已通过自动校验，但仍待独立实机验收。

  1. 01
    安装到“应用程序”

    打开 DMG，把 CoreRobin.app 拖进“应用程序”文件夹，然后推出磁盘映像。

  2. 02
    从“应用程序”启动

    从“应用程序”启动 CoreRobin。当前已签名并公证的版本通常可以直接打开。

  3. 03
    如果 macOS 仍然拦截

    最新版 macOS 安装包已经签名、公证并装订票据，通常不需要“仍要打开”。如果仍被拦截，请核对官方来源和 SHA-256，并报告 macOS、芯片及 CoreRobin 版本。

如果提示“CoreRobin.app 已损坏”

“已损坏”表示包体、签名或下载完整性异常。不要绕过；请重新下载最新版、核对 SHA-256，问题仍存在时再报告。
