COREROBIN 使用指南

第一次使用，从这里开始。

这份指南会带你看懂每个页面是做什么的，也会告诉你结束进程和清理文件时需要注意什么。

约 8 分钟读完
01

开始使用

第一次打开 CoreRobin，可以按这四个步骤熟悉界面。

先下载适合当前系统的安装包前往 GitHub Releases 下载。当前构建尚未配置平台发布签名或 Apple 公证；macOS 第一次打开若被阻止，请在“系统设置 → 隐私与安全性”中确认打开。Windows 与 Linux 安装包属于早期预览版本。Release 提供 SHA-256、SPDX SBOM 和来源证明，但它们不能替代平台签名。
  1. 1
    等几秒，让数据准备好

    刚启动时，网络和磁盘速率可能显示“预热”。等到下一次刷新就会出现正常数据。

  2. 2
    先看电脑状态

    日常模式会先告诉你电脑现在怎么样，并给出一个最值得做的动作；没有问题时不用逐项检查。

  3. 3
    从实际感受开始

    如果电脑卡、风扇响、耗电快、空间不足、开机慢或网络异常，打开“帮我解决”；需要进程树等详细信息时，再使用页面顶部的“进入专业模式”。

  4. 4
    选择语言和后台方式

    界面支持 10 种语言。设置中还可以让 CoreRobin 登录后静默启动，并控制 macOS Dock 图标和 Robin 桌面伙伴。

02

电脑状态与帮我解决

“电脑状态”只给出一个稳定结论和一个主动作；“帮我解决”则从你实际遇到的卡顿、发热、耗电、空间、启动或网络问题开始。持续问题确认后不会因一次正常采样突然消失，恢复也会经过稳定确认。技术数字按需展开。

◎先看结论

Robin 状态伙伴会停在当前检查阶段，并告诉你现在是否需要处理。

⌁从实际问题开始

不用理解术语，直接选择你正在遇到的情况。

↗跟着步骤处理

查看结论和相关应用，操作后再确认问题是否解决。

03

应用

应用不再是日常模式的一级页面。检查结果与应用有关时，才会出现一份不会每秒重新排序的稳定快照；CPU、内存和磁盘依据按需展开。专业模式提供进程树、搜索、排序和完整详情。

  • 单击应用或进程，可以查看它的位置、启动命令、上级进程和最近五分钟的占用。
  • 优先使用“请求结束”，让应用有机会保存状态；只在必要时强制结束。
  • 为了避免结束错进程，CoreRobin 会在执行前再次确认目标。进程已经退出或发生变化时，操作会自动停止。
04

存储

存储页显示每个磁盘还剩多少空间、最近的读写变化，以及哪些应用正在频繁读写。磁盘空间不足时，可以从这里直接进入“清理”页面。

macOS 系统卷只显示一次属于同一个 APFS 系统卷组的 / 与 Data 卷会合并显示，不会重复计算；外置磁盘不受影响。
05

清理

清理页完整扫描系统磁盘，再用扇形图显示空间被什么占用了。默认按真实文件路径展示，也可以切换到用途分类；扫描不会进入外置磁盘或其他挂载点。

macOS 完全磁盘访问权限首次扫描前会先说明用途并打开系统设置。如果列表里没有 CoreRobin，请点击列表下方的 + 并选择 CoreRobin.app；应用内可一键在访达中定位。授权后返回应用会自动继续，也可以选择只扫描当前可访问的位置。
单击查看里面拖进清理篮检查后再删除
只删除你认识、并且确定不再需要的内容不确定的内容先保留。可以重新生成的缓存适合作为起点，但缓存也不等于全部都能安全删除。
  1. 1
    等待扫描完成

    页面会持续显示正在扫描的位置、已经检查的项目和找到的空间。你也可以随时停止。

  2. 2
    查看大文件夹

    扇区越大，占用的空间越多。单击文件夹进入，单击中间区域返回上一级。

  3. 3
    放进清理篮

    按住扇区开始拖动，清理篮会固定显示在窗口底部。放进清理篮不会立即删除文件。

  4. 4
    刷新后永久删除

    CoreRobin 会只重扫清理篮中的目标并显示最新大小和文件数；发生变化时必须再次刷新并重新确认。文件不会进入废纸篓。

完整扫描结果会在本机保留 7 天用于浏览。缓存结果不能直接授权永久删除；打开删除复核时只会权威重扫清理篮中的目标。发现大小、文件数或深层内容变化后，旧确认失效，必须查看更新结果、再次刷新并重新勾选。

永久删除不会进入废纸篓删除后无法通过 CoreRobin 恢复。确认名称、位置和大小都正确后再继续。
06

网络

在日常模式中选择“网络好像有问题”后，CoreRobin 会先检查当前收发状态和活动连接。专业模式的网络页保留接口、流量和 TCP / UDP 连接详情。系统未显示应用名并不代表连接有问题。

07

启动项

日常模式只有在你选择“开机越来越慢”时才显示启动项检查。支持的第三方项目可以关闭并恢复；系统项目只供查看。CoreRobin 不会直接删除它们的配置文件。

08

记录与历史

  • 日常模式的“记录”用简单时间线说明电脑什么时候开始变忙、什么时候恢复正常；专业模式保留完整的资源历史与筛选。
  • 同一次问题的提醒和恢复会合并成一个事件，保留开始时间、持续时长、可能原因和最终结果。
  • 记录可以保留 1、7 或 30 天。你也可以关闭历史记录，或者随时清空。
  • 只有问题持续一段时间时才会提醒；同一问题不会反复弹出，稳定恢复后会补充一次恢复通知。CPU、内存和磁盘提醒可以分别关闭。
09

设置

日常模式设置提供 10 种界面语言、文字大小、动态效果、桌面提醒、本机历史和 Robin 偏好，也可以登录系统后静默启动；macOS 可选择是否显示 Dock 图标。需要更多技术信息时，可以使用页面顶部常驻的“进入专业模式”；专业模式设置继续提供采样速度、连接刷新、颜色提示和默认进程视图。

主窗口、状态栏和 Robin 使用同一健康状态关闭主窗口后，单击状态栏图标可开关紧凑面板，双击直接打开主窗口；登录时静默启动不会弹出启动画面或主窗口。
10

安全与隐私

结束进程应用会在最后一步确认，并再次检查目标。系统关键进程和 CoreRobin 本身不能被误关。
永久删除只处理你个人文件夹内的普通文件和文件夹。主目录、废纸篓本身、链接和其他磁盘不会被删除。
系统权限CoreRobin 不会自动申请更高权限。看不到某些信息时，页面会直接说明。
你的数据监控、文件、进程、历史和连接信息只保存在这台电脑上，不会上传或同步。扫描只读取名称、大小、位置等文件信息，不读取文件内容。
11

常见问题

刚启动时为什么没有网络或磁盘速度？

应用需要先后读取两次数据才能算出速度。等到下一次刷新就会显示。

为什么扫描要等很久？

扫描时间主要取决于文件数量和磁盘速度。大量小文件通常比少量大文件更慢。页面会持续更新进度，你也可以主动停止。

为什么大小和 Finder / Explorer 显示的不一样？

CoreRobin 显示文件实际占用的磁盘空间。压缩文件、稀疏文件和硬链接等情况，可能会让这个数字与文件标称大小不同。

还是遇到了问题？

可以在 GitHub 提交问题。写清楚你的操作系统、刚才做了什么，以及看到的错误信息，会更容易定位。安全问题请使用私密漏洞报告。
