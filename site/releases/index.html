平台覆盖

哪些平台经过了怎样的验证。

macOS · Apple Silicon推荐

M 系列 Mac 是主要实机验证路径；使用 aarch64 DMG。

macOS · Intel覆盖有限

提供 x64 DMG，并验证自动构建与包完整性；Intel 实机覆盖较少。

Windows · x64早期预览

自动构建与测试通过；真实设备、桌面集成和安装器覆盖仍有限。

Linux · x64早期预览

自动构建与测试通过；发行版、桌面环境和 WebKitGTK 组合尚未全面验证。

当前限制

下载前应该知道的事。

  • macOS 尚无 Developer ID 签名或 Apple 公证；Windows 尚无平台签名。
  • 温度、电池健康度、连接归属和部分启动项能力取决于操作系统与硬件；无法读取时会显示为不可用。
  • 网站只能根据浏览器推荐操作系统，不能可靠判断 Mac 芯片，也不会自动下载。
版本历史

v0.0.4

2026-07-17

安装可靠性与空间扫描体验修复

  • 修复 Apple Silicon DMG 中应用包签名不完整、可能被 macOS 判定为已损坏的问题。
  • 发布前会从 DMG 重新挂载验证完整签名、资源封装和目标架构。
  • 恢复空间扫描的彩色轨道、扫描扇区、呼吸光晕与节点脉冲。
  • 修正状态栏缩略面板高度，并增加小屏幕滚动兜底。
  • Apple Silicon Mac 即使临时运行 Intel 包，也能补齐电池健康度；原生包仍是温度读取的推荐路径。
  • 下载页明确区分操作系统推荐与 Mac 芯片选择，并补充两类 macOS 安装提示。
