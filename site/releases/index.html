平台覆盖

各平台当前的验证情况。

macOS · Apple Silicon推荐

M 系列 Mac 是主要实机验证路径；使用 aarch64 DMG。

macOS · Intel尚待实机验收

已签名、公证并装订票据的 x64 DMG 通过自动校验；仍待 Intel 实机验收。

Windows · x64早期预览

自动构建与测试已经通过，但真实设备、桌面集成和安装流程的验证仍然有限。

Linux · x64早期预览

自动构建与测试通过；发行版、桌面环境和 WebKitGTK 组合尚未全面验证。

当前限制

下载前应该知道的事。

  • macOS DMG 已使用 Developer ID 签名、经过 Apple 公证、装订票据并通过 Gatekeeper 校验。Apple Silicon 已完成实机验证；Intel 已通过自动校验，但仍待独立实机验收。
  • Windows 与 Linux 仍是早期预览版本，尚无平台签名，也未分别完成真实设备验证。
  • 温度、电池健康度、连接归属和部分启动项能力取决于操作系统与硬件；无法读取时会显示为不可用。
  • 网站只能根据浏览器推荐操作系统，不能可靠判断 Mac 芯片，也不会自动下载。
版本历史

v0.1.9

2026-07-21

网络趋势与日常体验完善

  • 打开网络质量页会自动检查，页面停留期间每 30 秒采样，展示最近 15 分钟的延迟、抖动和丢包趋势。
  • 首次扫描前，清理页会展示只读范围和明确的开始按钮。移到废纸篓仍是可恢复的默认方式，直接删除需要单独确认。
  • 设置现在使用一致的卡片布局与控件。专业模式默认选择有用的进程，应用关注规则也改用统一的应用、指标、阈值和持续时间字段。
  • macOS DMG 已使用 Developer ID 签名、经过 Apple 公证、装订票据并通过 Gatekeeper 校验。Apple Silicon 已完成实机验证；Intel 已通过自动校验，但仍待独立实机验收。
版本历史

v0.1.8

2026-07-20

彻底卸载与真实应用身份

  • 应用成为两种模式中的一级入口，显示本地化名称和真实图标，复用缓存清单，并在 macOS 上提供逐路径复核的彻底卸载助手。
版本历史

v0.0.4

2026-07-17

安装可靠性与空间扫描体验修复

  • 修复 Apple Silicon DMG 中应用包签名不完整、可能被 macOS 判定为已损坏的问题。
  • 发布前会重新挂载 DMG，检查应用签名、资源封装和目标架构。
  • 恢复空间扫描的彩色轨道、扫描扇区、呼吸光晕与节点脉冲。
  • 修正状态栏缩略面板的高度，并让较小屏幕可以滚动查看完整内容。
  • Apple Silicon Mac 临时运行 Intel 包时也能读取电池健康度；如需读取温度，仍建议使用原生包。
  • 下载页现在会分别说明操作系统推荐和 Mac 芯片选择，也区分“无法验证开发者”与“应用已损坏”两类提示。
